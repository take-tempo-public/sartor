The prompt-tuning loop A/Bs a candidate system prompt against the baseline
      without editing the persona constants (the analyzer.prompt_overrides()
      primitive). Candidate runs log prompt_version=candidate:<hash> so they never
      pollute score-over-time.

    
The A/B is now interactive on this tab: pick a constant, edit its text,
      and run candidate-vs-baseline in the browser. The irreversible promote (edit the
      analyzer.py constant, bump PROMPT_VERSION, log
      TUNING_LOG.md) stays a human/agent step — the
      /prompt-tune and /tune-from-annotations skills automate
      drafting + promotion end to end.
