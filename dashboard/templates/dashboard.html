The prompt-tuning loop A/Bs a candidate system prompt against the baseline without editing the persona constants (the analyzer.prompt_overrides() primitive). Candidate runs log prompt_version=candidate:<hash> so they never pollute score-over-time.

This tab is a read-only scaffold. The interactive loop — drafting a candidate, running candidate-vs-baseline, promoting on approval — runs today via the /prompt-tune and /tune-from-annotations skills and lands as an in-console surface in a later, sign-off-gated branch.
