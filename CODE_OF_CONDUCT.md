# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1, as its code of conduct.

The full text is published at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

A plain-text version is also available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt>

## Reporting

Concerns about conduct in this project's spaces (issues, pull requests, discussions, code review) should be reported privately via [GitHub Security Advisories](https://github.com/Cooksey/resume/security/advisories/new) — the same channel as security issues. Reports are read by the project maintainer.

The maintainer will respond within 5 business days, keep the report confidential, and follow the enforcement guidelines in the linked Contributor Covenant text.

## Scope

This applies to behavior in all project spaces — the repository, its issues and PRs, GitHub Discussions, and any other forum where someone is representing the project.
